# Private Beta Guide

Use this app with 3-5 trusted testers before adding more stores or sharing widely.

## Tester instructions

1. Open the Railway link on a phone.
2. Run one SKU search, for example `HF4198-001`.
3. Run one keyword search, for example `Black Cat 4`.
4. Open at least one store link.
5. Track one product or release.
6. Submit feedback in the Beta tab.

## Admin instructions

Before inviting testers:

- Run Beta Readiness.
- Run Store Check.
- Run Production Monitor.
- Confirm Lookup Unavailable is not counted as Sold Out.
- Confirm release calendar contains future releases only.

After testing:

- Export logs.
- Review failed searches.
- Review store health.
- Disable stores that repeatedly fail without useful fallback states.

## Scaling rule

Do not add more stores until current stores are stable. Expand in small batches and review health after each batch.
